关于hexo建立个人blog

  1. 使用npm install hexo-cli -g 创建hexo环境
  2. 使用 hexo -v 查看是否安装成功，成功后会有相应的版本号
  3. hexo init blog 在任意路径建立个人blog
  4. npm install 再次安装npm 早版本的命令 现在可以省略这一步
  5. hexo server 启动hexo 服务器

创建一个新的blog

使用git进入 blog文 件夹,输入 hexo new fiel file 你需要建的文件
该文件会在 source > _post 下面自动创建一个你新建的 file.md 的文件，该文件会在主页自动显示.
